# Scramble Text
React component library for scramble text effect

## Usage
```jsx
import { ScrambleText } from "@shahriyardx/scramble-text"

// in your component
<ScrambleText
   text="SCRAMBLE"
   speed={30}
   iterationPerCharacter={3}
   characters="ABCDEFGHIJKLMNOPQRSTUVWXYZ"
   enabled={true}
>
```
